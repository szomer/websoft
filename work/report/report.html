S02

I have worked with HTML and CSS before. From my understanding, HTML is used to create the elements of the webside shown to the user and create a structure. For example all the pictures, text, title, divisions and so on. CSS is used to style all of those elements. For example to change the color, size, space of these elements. I have not worked with javascript before however I believe I understand it's purpose. Javascript is the code running behind the scenes and is not visible for the user to see. For example if a website uses specific animations, calculations, processes information and so on.

What happens when a computer enters the website "http://website.com/index.html" into the browser? First of all, the user will be making use of the HyperText Tranfer Protocol. This protocol is used to connect the browser to a webserver. With a HTTP resonse from the server we receive the html document and are able to view the website.

It has been multiple years since I worked with html again and I started remembering the keywoards and the use of HTML, CSS adn Javascript again.
